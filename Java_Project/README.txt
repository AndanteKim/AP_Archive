To run the program:
1. Decompress the group_7_assignment7.pde
2. Open up group_7_assignment7.pde
3. Ensure that there are the following classes: Bullet, Enemy, EnemyField, Player, Shield, ShieldSegment, Status
4. Press play button
To play the game:
1. Use the ‘A’ and ‘D’ keys to move the spaceship right and left respectively.
2. Use the spacebar to shoot at the Enemies. 
3. The goal is to shoot all of the enemies down. 
4. The player loses if they get hit three times or the enemies pass the spaceship.
